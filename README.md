## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML
- CSS
- JS

## 💻 Projeto

Este é um exercício ensinado durante o curso Explorer da [Rocketseat](https://www.rocketseat.com.br/).

No projeto Cálculo de IMC o usuário informa seu peso e altura para obter o seu IMC.

Alguns conceitos aplicados no projeto:

- Tags HTML semânticas
- CSS:
  - Grid
  - Transition
  - Variáveis
- JS:
  - Funções
  - Classes
  - Manipulando a DOM
  - Ouvindo Eventos da DOM
  - Variáveis
  - Objetos
  - Validação de dados
  - ES6 Modules
  - Modularização de componentes da aplicação
  - Regras básicas de Clean Code
